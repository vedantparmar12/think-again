## üöÄ Quick Start

### Installation

```bash
# Clone the repository
git clone https://github.com/vedantparmar12/think-again.git
cd think-again

# Switch to the latest release
git checkout release/v2.1.0

# Install skills (Windows PowerShell)
Copy-Item -Recurse .codex/skills/* "$env:USERPROFILE\.codex\skills\"

# Install skills (macOS/Linux)
cp -r .codex/skills/* ~/.codex/skills/

# Restart Codex
exit
codex
```
